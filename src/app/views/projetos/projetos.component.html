Esse projeto faz parte da NLW e-Sports trilha Ignite promovido pela RocketSeat durante imersão NLW esports. Para a construção deste sistema foi utilizado a biblioteca react, e as demais tecnologias: HTML, CSS, JavaScript, TypeScript, ReactJS, TailwindCSS, com o objetivo de desenvolver um programa capaz de encontrar duos de jogos.

Este projeto foi desenvolvido com base na aula CodeExpress, promovida por Rodrigo Lucas, no canal Jovem programador. Para a construção deste sistema foi utilizado a biblioteca react, e as demais tecnologias: HTML, CSS e JavaScript.
